问题

详细描述问题的范畴/条件/上下文/…

如何迅速在 Mac OS X 上安装好可用好用的 julia 环境。

解决方案的几点要求

  • 简单，不容易掉坑；
  • 最好可以用熟悉的开发环境；
  • 最好跨平台，经验可以复用；

所以选择全平台，容器化的应用 docker。控制 docker 的客户端使用 vscode 的插件，容器内开发使用 vscode 的远程模式。
