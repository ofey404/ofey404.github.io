My ‘Only One OS X shortcut’: ⌘-⇧-/

For English writing training, also preaching my favorite OS X shortcut.

Published on Mar 2, 2020
  • OSX

The great Richard Feynman said in his lectures of physics:

If, in some cataclysm, all of scientific knowledge were to be destroyed, and only one sentence passed on to the next generation of creatures, what statement would contain the most information in the fewest words?

OS X applications often have consistent, well-arranged shortcut sets leaded by the system-wide ⌘(command) key. The consistency distinguishes the OS X shortcut design from Windows and Linux designs. There is even a specific application to help you to master the shortcuts.

If all my mastering of OS X shortcuts have been gone in some cataclysm, just like Feynman described, except one shortcut, which should I keep? I meditated in silence, between ⌘-SPACE (spotlight) and ⌘-⇧-/ (Search for help), finally I choose the ⌘-⇧-/ as my best bet.

⌘-⇧-/: instant help search

While hitting ⌘-⇧-/ in any mac app, help menu would popup from the menu bar, and the input focus to the search box instantly. Then you can use arrow keys to navigate through search results.

By typing the name of operation you want, anything that you can do by shortcuts or menubar buttons is within your reach.

For example, when you open Finder App, and want to go to Applications Folder, you can directly hit ⌘-⇧-/, type in “application” in the popup (actually just type ”ap” the result has been shown), then hit the down button and enter.

Fuzzy search is also supported, eg: hit ⌘-⇧-/ and input “t w l”, you can Tile Window to Left of Screen without reaching your mouse.

The workflow is just like VSCode’s “Show All Commands” (F1 or ⌘-⇧-/ in OS X), you just type in what you want to do. It works like a tiny command palette. If you are used to VSCode’s F1 or any application launcher, I bet you’ll like it.

Why bother typing?

Less Keystroke, all in one, determinism, and a visual learning experience to keyboard shortcuts.

Keystroke sequence of ⌘-⇧-/ are short enough at most situation. ⌘-⇧-/ is just like an arsenal of all menu features within a few keystrokes, not only those operations bound with keyboard shortcuts, but also all other operations.

To Click a menu bar button with your mouse, you must find the button on screen before click it, so that it’s almost impossible to form a muscle memory(probably the No.1 criticism of Mouse & buttons from keyboard lovers).

With ⌘-⇧-/, muscle memory can be built. Searching in instant help popup don’t need to leave your keyboard, and the keystroke sequence of a certain function is determined.

If you use some function really often, it won’t be a bother to memorize its unique keyboard shortcut.

Last but not least, ⌘-⇧-/ provide an interactive learning experience of learning shortcuts. While moving cursor on an operation in search results, there will be pretty interaction effect. The effect would indicate you where the function is, and the keyboard shortcut of it(if shortcut exists).
