macOS X 设置长按连续输入

Posted on 2020-01-20

When I gonna do a sled slide with vim keys, the phonetic symbol function of OSX stops me.

关闭 macOS X 10.14 长按元音字母键输入音标的功能，在命令行下使用 vim 的时候经常碰到。

做个备忘。参考了macOS 长按连续输入的简单设置方法 ziheng’s Blog，感谢！

实际上，苹果「Apple」已经在用户系统配置项「Mac OS X User Defaults System」里预留出了长按连续输入的设置选项。我们可以方便地使用defaluts命令行工具关闭ApplePressAndHoldEnabled功能，设置完成后，注销或重启使其生效即可。
